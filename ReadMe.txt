#C Language Basics:
-------------------
- C structure oriented programming Language also called "Procedure oriented programming Language".
- Developed at Bell Laboratories in 1972 by Dennis Ritchie.
- C has been coded in assembly language.
- Program structure follows "Top Down Approach"
- Middle level language used in general-purpose programming.

- Feature of C Language : Reliability, Portability, Flexibility, Interactivity, Modularity, Efficiency and Effectiveness.

- Usage : Compilers and Assemblers, Database Systems, OS development, Network drivers.
    Database Systems -> DBMS (Database Management System). create, retrieve, update, and manage data in a database.

- C programming is a case sensitive programming language.


- Disadvantage : C does not provide Object Oriented Programming (OOP) concepts.
